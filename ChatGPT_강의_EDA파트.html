            dcm.RescaleType if 'RescaleType' in dcm else None]

Pydicom 라이브러리를 이용해 모달리티 및 VOI(Visual Object Image) lookup 테이블을 이용하여 픽셀 값을 조정하는 코드를 작성하겠습니다.

python
import pydicom
from pydicom.pixel_data_handlers.util import apply_modality_lut, apply_voi_lut

# DICOM 파일 로드
dcm = pydicom.dcmread('파일 경로')

# 'Modality' lookup table 적용
data = apply_modality_lut(dcm.pixel_array, dcm)

# 'VOI' lookup table 적용
data = apply_voi_lut(data, dcm)

# 결과 확인
print(data)

각 코드의 설명은 다음과 같습니다.

  1. pydicom.dcmread('파일 경로') : 지정된 경로의 DICOM 파일을 읽습니다.
  2. apply_modality_lut(dcm.pixel_array, dcm) : 'Modality' lookup table을 사용하여 픽셀 값을 변환합니다. 이 함수는 DICOM 데이터셋의 Rescale Slope 및 Rescale Intercept를 이용하여 픽셀 값을 조정합니다.
  3. apply_voi_lut(data, dcm) : 'VOI' lookup table을 사용하여 픽셀 값을 변환합니다. 이 함수는 DICOM 데이터셋의 Window Center 및 Window Width를 이용하여 픽셀 값을 조정합니다.
